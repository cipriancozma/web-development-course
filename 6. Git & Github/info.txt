Q. What is Version Management?
A way to control and track code changes over time

MacOS Terminal (z Shell):
pwd - print working directory
cd -  change directory
ls - list items
clear - clears the current view of the terminal
mkdir - make a new directory
touch - creates a new file (touch dummy.txt)
rm - remove file
rmdir - remove empty directory

Windows Command Prompt (cmd):
cd - change directory
dir - to list 
cls -clear window
mkdir - make directory
echo - create files
rmdir - remove directory
del - delete file

Q. Git - Initializing Repo & Creating commit
git init - initialize a repository
git status - info about our current git project
git add <filename>- to add files 
git commit -m - to add a message to our modifications
git push - to push our commit to remote
git log - list of commits

Q. Understanding Branches, Merging & Fixing Merge Conflicts
git branch -a -> show the branches we have created
git branch -m -> rename the branch 
git checkout -b -> create and checkout on a new branch
git reset --hard HEAD~1 -> reset one commit back our branch
git branch -D <filename> -> delete branch
