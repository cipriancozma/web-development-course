Q. Flexbox
- allows us to manage the space or the way elements move inside a box and define the layout in one dimension

Q. Understanding Fixed & Absolute positioning
Position fixed -> the element remains fixed when you scroll in the document
Position absolute -> removes the element from page flow and after that you can control it.

Q. Understanding Parent - Child Margin Collapsing
Can be solved by adding a padding to the parent.

Q. GRID
The GRID system means to define our layout in two dimensions, columns and rows.

Q. Position
Position absolute - takes an element out of the document flow and we can move it by simply adding top, bottom, left or right
Position relative - is relative to its current position 
Position fixed - takes the element out from the document flow and keeps it fixed
Position static - is the default position
